# ❤️ HeartCare AI

An **Intelligent Heart Attack Risk Prediction System** built using **Machine Learning** and **Streamlit** to help users assess heart attack risk based on various health parameters.

---

## 📌 Project Overview

HeartCare AI is a Machine Learning-based healthcare application that predicts the risk of heart attack using a trained **Logistic Regression** model. The application provides an interactive interface where users can enter their health details and receive an instant prediction along with personalized health insights.

This project was developed as part of my learning journey to understand not only Machine Learning model deployment but also different AI development approaches and healthcare application design.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/HeartCare-AI.git
```

Move into the project folder

```bash
cd HeartCare-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
